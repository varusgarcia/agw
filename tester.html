Continuity Tester

The slider controller is a simple solution for powering and controlling the slider. It offers basic functions such as changing the speed and direction for shooting video and some more advanced for timelapse. For this mode, and with the help of a intervalometer, the controller allows for a more stabilised timelapse, because it won’t move while taking the photo. This also allows to take much longer timelapses.

What you need:

A simple Cardboard Box

If you can't find one, do it yourself by gluing six cardboard pieces toghether like i did.

2X AA Batteries

These are going to power the tester, so make sure they are full.

Some Wire

You will really only need three pieces, two long and one short.

LED

The LED is going to be the visual detector, so make sure it shines bright.

What can it do?

It really only has one task: Detect if a circuit is closed.
It might seem too simple to even start making it, but trust me, it is really usefull and will save you so much time in checking wires.

How it works

Let's see how "everything" is wired:

(Clic on the video to pause/play it)

As you can see, it is really easy. It's a basic circuit (LED to Battery) interrupted by two cables. When they touch, the circuit closes and the LED lights up.
It's, as said, a really usefull and simple tool that will help you figure out how something is wired or if something is broken.
